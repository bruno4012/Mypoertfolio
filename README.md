# Bruno Martins — Portfolio Website https://bruno4012.github.io/Mypoertfolio/

Personal portfolio site for Bruno Martins, IT Specialist & Frontend Developer based in Dublin.

## Project Structure

```
website portfolio/
├── index.html    # Main HTML structure
├── styles.css    # All CSS styles
├── script.js     # JavaScript (modal logic, scroll animations)
└── README.md     # This file
```

## Sections

- **Hero** — Introduction and call-to-action
- **About** — Background, skills and experience summary
- **Experience** — Career history with clickable modals for each role
- **Education** — Academic qualifications
- **Contact** — Phone, email, LinkedIn, and portfolio links



## Tech Stack

- Vanilla HTML, CSS, JavaScript
- [Google Fonts](https://fonts.google.com/) — Playfair Display, DM Sans, Space Mono
- [Font Awesome 6](https://fontawesome.com/) — Icons
